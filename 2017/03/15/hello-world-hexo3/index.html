Welcome to Hexo! This is your very first post. Check documentation for more info. If you get any problems when using Hexo, you can find the answer in troubleshooting or you can ask me on GitHub. Deployment settings has been changed and Chinese friends may need VPN to complete hexo init or npm install or even hexo deploy, watch out that hexo-deployer-git and hexo-generator-feed need to install separately and hexo-generator-sitemap may cause error when run hexo g.

Update

My way is npm update -g hexo-cli, hexo init newblog, cp newblog/package.json oldblog/, watch out if you have changed oldblog/package.json, try vimdiff oldblog/package.json newblog/package.json first, and the last hit cd oldblog; rm -rf node_modules; npm install.
